優惠分享助理 Privacy Policy

最後更新日期：2026 年 5 月 30 日

1. 應用程式簡介

「優惠分享助理」（以下簡稱「本應用程式」）是一款供個人使用的自動化工具， 用於從 Coupang 電商平台發掘特價商品，並透過 Threads 平台分享優惠資訊。 本應用程式由個人開發者維護，不對外公開商業服務。

2. 我們蒐集的資料

本應用程式在連結 Threads 帳號後，將存取以下資料：

  • 帳號基本資訊：Threads 使用者 ID、使用者名稱
  • 發文資料：由使用者主動輸入並確認後，才透過 Threads API 發佈的貼文內容

本應用程式不蒐集以下資料：

  • 密碼或登入憑證
  • 個人財務或信用卡資訊
  • 裝置識別碼或位置資訊
  • 通訊錄或私訊內容

3. 資料使用目的

蒐集的資料僅用於以下目的：

  • 驗證 Threads API 存取權限，確認帳號連線有效
  • 依使用者指示，透過 Threads API 發佈貼文

我們不會將任何資料出售、轉讓或提供給第三方用於商業分析或廣告目的。

4. 資料儲存與安全

Threads API Access Token 僅儲存於本機伺服器環境設定檔（.env）， 不會上傳至任何外部伺服器或雲端服務。 本應用程式部署於個人本機或私有伺服器，僅供授權使用者存取。

我們採取合理的技術措施保護資料安全，包括：

  • API 金鑰存放於環境變數，不寫入程式碼
  • 伺服器不對外公開存取
  • 定期更新 Access Token

5. 資料保留期限

發佈的貼文記錄（包含文案內容與發文時間）儲存於本機資料庫， 供使用者自行查閱效益數據。 使用者可隨時自行刪除資料庫中的相關記錄。 應用程式停止使用後，本機資料庫資料不會自動刪除，需由使用者手動清除。

6. 第三方服務

本應用程式使用以下第三方服務，各服務有其獨立的隱私政策：

  • Meta Threads API：用於貼文發佈。 詳見 Meta 隱私政策
  • Coupang Partners API：用於產生分潤連結。 詳見 Coupang 隱私政策

7. 使用者的資料權利

你有權：

  • 撤銷授權：隨時透過 Threads 應用程式設定撤銷本應用程式的 API 存取權
  • 刪除資料：要求刪除儲存於本應用程式的所有相關資料
  • 查閱資料：要求查閱本應用程式所儲存的資料內容

如需行使上述權利，請透過下方聯絡方式與我們聯繫。 我們將在 30 天內回應你的請求。

8. 資料刪除請求

依據 Meta 平台政策，使用者可要求刪除與本應用程式相關的所有資料。 提出刪除請求後，我們將於 30 天內完成刪除，並以私訊通知確認。

請透過 Threads 私訊 @rec.8888 提出刪除請求。

9. 隱私政策變更

本隱私政策如有重大變更，將更新頁面頂端的「最後更新日期」。 建議使用者定期查閱本頁面。 繼續使用本應用程式視為接受更新後的隱私政策。
